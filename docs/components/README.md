# Component Documentation

This folder contains component-level documentation for the Be-Honest app. Each file documents a single component: its purpose, props, usage examples and implementation notes.

Files in this folder:

- `Feed.md` — Main feed showing posts and handling realtime updates.
- `FeedHeader.md` — Controls/filters for the feed.
- `Hero.md` — Top hero area on the home page.
- `LeftCol.md` — Left column with quick post creation and filters.
- `RightCol.md` — Right column showing trending content and additional info.
- `PostContent.md` — Renders the body/content of a post.
- `PostActions.md` — Like/share buttons and actions for a post.
- `ShareCard.md` — Share preview card used in the UI.
- `ContactForm.md` — Contact page form component.
- `Navbar.md` — Top navigation and brand.
- `ThemeProvider.md` — Theme context/provider used by the app.
- `Filters.md` — Reusable filter UI elements.
- `GradientCard.md` — Small UI card used in the `ui/` folder.
- `LoginModal.md` — Login modal (mock) used for demonstration.
- `MobileLeftColToggle.md` — Mobile toggle control for left column.
- `ShoutLink.md` — Shout link submission/preview component.
- `ClientLoaderWrapper.md` — Wrapper to safely render client-only content.
- `GlobalLoader.md` — Global loading UI used during data fetches.
- `Footer.md` — Application footer.
- `PageId.md` — Small utility component for page id display.

If you want more fields (prop tables, storybook examples, or TS types), tell me which components to expand and I will add them.
